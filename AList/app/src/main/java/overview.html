Displays a list of items each with a name, description, category and status from a database.

Includes a main activity to display a list of {@link myapp.alist.datarepository.db.ListEntity} items populated from a database. Help and about activities display those respective details. An alert dialog fragment is used for a delete warning.

Includes a {@link myapp.alist.datarepository.viewmodel.ListViewModel}, {@link myapp.alist.datarepository.datarepository.DataRepository}, {@link myapp.alist.db.AListDatabase}, {@link myapp.alist.db.ListDao} and a {@link myapp.alist.db.ListEntity} to insert, select, update and delete rows in a list table in the database.
